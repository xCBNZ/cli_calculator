# CLI Calculator in C

Simple **Command-Line Calculator** written in **C** that supports basic arithmetic operations.

## Features

- Addition (`+`)
- Subtraction (`-`)
- Multiplication (`*`)
- Division (`/`)
- User-friendly command-line interface

## Installation

1. Clone the repository:
   git clone https://github.com/xCBNZ/cli_calculator.git
   cd cli_calculator
   
2. Compile with gcc:
   gcc -o calculator cli_calculator

3. Run:
   ./calculator
